# CFB Updates Website

Your custom college football dynasty league news site.

## File Structure

```
cfbupdates/
├── index.html          ← Main homepage
├── articles/           ← Individual article pages go here
│   └── article-template.html  ← Copy this for each new article
└── images/             ← All screenshots/images go here
```

## How to Upload to Porkbun

1. Go to your Porkbun Web Hosting management panel
2. Look for "File Manager" or "FTP Access"
3. Upload all these files to the `public_html` or root directory
4. Make sure to maintain the folder structure (articles/ and images/ folders)

## Adding a New Article

### Step 1: Send me your game info
- Screenshot(s) of the game/box score
- Final score
- Key stats (passing, rushing, receiving leaders)
- Any notable plays or storylines

### Step 2: I'll generate
- A complete article HTML file
- Updated index.html with the new article card added

### Step 3: You upload
- Put the article file in `/articles/`
- Put any images in `/images/`
- Replace your `index.html` with the updated version

---

## Quick Reference: Article Naming

Use lowercase, hyphenated names:
- `ohio-state-vs-michigan-week5.html`
- `texas-upsets-alabama.html`
- `playoff-semifinals-recap.html`

## Image Naming

Match images to articles:
- `ohio-state-vs-michigan-week5.jpg`
- `texas-upsets-alabama-thumb.jpg`
